# Music Visualization Research

Research materials for improving Fractured Jukebox visualizer.

## Directory Structure

```
research/
├── README.md              # This file - index of all research materials
├── notes.md               # Synthesis and analysis notes (key insights extracted from papers)
├── bibliography.md        # Comprehensive bibliography with quality rankings
├── raw-docs/              # Downloaded academic papers (PDFs)
└── references/            # Saved web pages and technical references
    └── fractals/          # Fractal mathematics reference materials
```

## Quick Start

1. **New to the project?** Start with `notes.md` for synthesized insights
2. **Looking for citations?** See `bibliography.md` for ranked paper list
3. **Need technical details?** Check `raw-docs/` for full PDFs or `references/` for saved web pages
