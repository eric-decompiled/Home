# Keyboard & Gesture Shortcuts

## Desktop

| Shortcut | Action |
|----------|--------|
| `Space` | Play/pause |
| `<` | Previous/rewind |
| `>` | Next track |
| `{` | Previous playlist |
| `}` | Next playlist |
| `/` | Search all songs (vim-like) |
| `?` | Show shortcuts help toast |
| `↑/↓` | Navigate song picker (when focused) |
| `Enter` | Select song (when picker open) |
| `Esc` | Close picker/search |
| `⌘/Ctrl + click` | Toggle theory bar |

## Mobile

| Gesture | Action |
|---------|--------|
| Tap | Play/pause |
| Long-press (500ms) | Toggle theory bar |

## iOS Landscape

| Gesture | Action |
|---------|--------|
| Tap anywhere | Show top bar (auto-hides after 2.5s) |
